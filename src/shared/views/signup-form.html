Sign-up form
In this page, you can experience account creation through a form based sign-up flow using a username and a password. ## How to use Simply enter a username and a password twice to create an account. *The password won't be saved to the server on this website. You don't need to create an account before signing in.* ## How to develop Use `input` tag for a `username` field and `password` fields. Append `autocomplete="username"` attribute for `username` field. Append `autocomplete="new-password"` attribute for `password` fields. ### Learning resources * [Sign-up form best practices](https://web.dev/articles/sign-up-form-best-practices)
