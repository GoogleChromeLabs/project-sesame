Password authentication
In this page, you can experience an ordinary password-based sign-in and passkey automatic upgrades. If you are at this page, it means you have entered a username in the previous page, or you are redirected here because you are trying to access a sensitive page that requires reauthentication. ## How to use Simply enter a password to proceed. Since the password won't be stored on this website, you can use a random text. After successful password authentication, the password manager creates a new passkey automatically for you. This is done by passkey automatic upgrades. ## How to develop ### passkey automatic upgrades When you sign in with a password, the password matches the one stored to the password manager and no passkeys are created yet, the password manager can automatically create a new passkey. This helps passkey adoption by your users. * [Automatically create passkeys for your users using Conditional Create](https://web.dev/articles/webauthn-conditional-create)
