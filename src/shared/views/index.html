Index page
This is Project Sesame's index page. Project Sesame is designed to showcase one authentication demo at a time. By visiting this page, users can get an overview of available authentication demos. Links in our developer documentation usually point at a specific authentication demo (as a path) and when a user signs out, they are brought back to where they sign in from.
