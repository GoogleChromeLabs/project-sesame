Unified sign-in
In this page, you can experience and learn unified sign-in flow using passwords and passkeys. This uses Credential Manager for the Web. ## How to use Click on the **Sign-in** button and a list of password accounts and passkey accounts will appear in a dialog. Choose one of accounts and proceed. ### Requirements * Chrome 135 or later with `chrome://flags#enable-experimental-web-platform-features` enabled. ## How to develop With Credential Manager for the Web, you can request `navigator.credentials.get()` with `PublicKeyCredential` (passkeys) and `PasswordCredential` (passwords) at the same time. In the future, `IdentityCredential` (FedCM) can be requested as well.
