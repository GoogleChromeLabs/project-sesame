New password creation
In this page, you can experience and learn a new password creation experience. ## How to use This is part of an identifier first sign-up experience. You must have entered a username in the previous page. Simply enter a password to proceed. After a successful password verification, you'll be signd in. *This demo doesn't save the password to the server. You can use a random text as your password.* ## How to develop Use `input` tag and `autocomplete="new-password"` attribute to the `password` field.
