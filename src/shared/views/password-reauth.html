Password reauthentication
In this page, you can experience and learn password-based reauthentication. ## How to use You are redirected here because you tried to access a sensitive page that requires reauthentication. Simply enter your password to proceed. After a password verification, you'll be signed in. *Since this demo doesn't save the password to the server, you can sign in with a random text as your password.* ## How to develop Require reauthentication after a certain duration has passed since the last credential verification. Pass the original destination to the server. After successful reauthentication, redirect the user back to the original destination. In this demo, the URL is passed as a query parameter, but the redirect logic should only accept same origin, to prevent open redirects.
